Este repositório contem arquivos das praticas de laboratório
da disciplina DCA0204 - ESTRUTURAS DE DADOS.


Os objetivos deste repositório são:

- controle de versão dos códigos
- acesso de não alunos da disciplina as atividades desta


Estrutura do Repositório

O padrão que estou tentando seguir é: para cada prática de laboratório existe 
uma pasta que contem duas pastas chamadas original, e modificado. A pasta original
contem o que nos é fornecido para iniciar o laboratório, e a modificado apresenta
as modificações feitas nos arquivos.


Obs:

1) Eu não sou acostumado a versionar códigos em C++ e em Java, logo recomendo que 
seja verifique o que esta sendo ignorado no .gitignore em caso de algo que deveria esta 
presente esta faltando.

2) Não sei se é uma boa prática que esse repositório contenha todas as 
atividades, não sei se ficará inconveniente por algum motivo.
